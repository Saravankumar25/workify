# Workify Document Style Guide

## Resume Formatting
- Use `#` for candidate name as the main heading.
- Use `##` for section headers (Summary, Skills, Experience, Education, Projects, Certifications).
- Use `**bold**` for job titles, company names, and degree names.
- Use `-` bullet points for experience descriptions and skills.
- Use `|` pipe separators for inline metadata (e.g., **Role** | Company | Date).
- Keep line lengths reasonable for PDF rendering.

## Cover Letter Formatting
- No heading — start directly with the greeting.
- Use `Dear Hiring Manager,` if the recruiter name is unknown.
- Separate paragraphs with blank lines.
- Sign off with `Sincerely,` followed by the candidate's name.

## General
- No emojis.
- No colored text or HTML tags.
- Professional, clean, and scannable.
- Optimised for ATS parsing — avoid tables, columns, or complex layouts.
